Waze User Churn Prediction

 Overview
This project predicts monthly user churn for Waze using machine learning. Identifying high-risk users allows for proactive retention strategies.

Key Features
- Feature Engineering: Developed custom metrics like `km_per_driving_day`.
- Models: Compared Random Forest and XGBoost classifiers.



XGBoost is the champion model, successfully identifying 87 true churners.
